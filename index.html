Games

Kakatte Koi Yo!

Kakatte Koi Yo! (Bring it On!) is a multiplayer fighting game that features cat characters that battle with fish. It has been featured at E3 in the College Game Competition as finalists. it has been released on itch.io and will be coming to Steam soon.

Hellevator

Hellevator is a VR fighting game where the player armed with a sword-gun hybrid battles waves of demons at each level of the arena turned elevator.
It was created for the Oculus Rift and features 2 unique enemy types with multiple animations, a shooting and slashing mechanic, and haptic feedback.

Banggora

Banggora is a game created for the Tiny Planet game jam. It is a 3D wave shooter game where the player traverses a tiny planet and shoots spinning carrot enemies. It was created in 6 days in a team of 2 people and ranked #36 out of 273 entries.

Pollen Nation

Pollen Nation is an arcade game created for a 2 day game jam.

i was a star

i was a star is a minimalistic collection game created for a 3 day game jam.
